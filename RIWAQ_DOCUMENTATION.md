# منصة رِواق - Riwaq Educational Platform

## نظرة عامة
منصة رِواق هي منصة تعليمية عربية متخصصة في تقديم دورات فيديو احترافية عالية الجودة.

## الألوان والهوية البصرية

### الألوان الأساسية
- **اللون الأساسي (Primary)**: `#3B2F82` - بنفسجي داكن
- **اللون الثانوي (Secondary)**: `#6467AD` - بنفسجي فاتح
- **اللون التمييزي (Accent)**: `#F3BD32` - أصفر ذهبي
- **الخلفية (Background)**: `#FFFFFF` - أبيض

### الخطوط
- **الخط الأساسي**: Cairo (Google Fonts)
- يدعم اللغة العربية بشكل كامل
- أوزان متعددة: 300, 400, 500, 600, 700, 800

## الصفحات المتوفرة

### 1. الصفحة الرئيسية (Home Page)
**الوصول**: `#home` أو الصفحة الافتراضية

**المحتويات**:
- قسم البطل (Hero Section) مع مقدمة المنصة
- عرض القيمة المقترحة
- أزرار CTA: "انضم الآن" و "استكشف الدورات"
- الدورات المميزة (3 دورات)
- قسم رؤية المنصة
- تذييل مع الشعار ومعلومات التواصل

**المميزات**:
- تصميم متجاوب (Desktop & Mobile)
- إحصائيات المنصة التفاعلية
- بطاقات دورات قابلة للنقر

---

### 2. كتالوج الدورات (Courses Catalog)
**الوصول**: `#courses`

**المحتويات**:
- شبكة عرض الدورات (Grid/List)
- شريط بحث متقدم
- فلاتر التصنيفات (البرمجة، التصميم، التسويق، الأعمال)
- 9 دورات متنوعة

**معلومات كل دورة**:
- صورة مصغرة
- عنوان الدورة
- وصف قصير
- السعر
- عدد الدروس
- المدة الزمنية
- زر "عرض التفاصيل"

**المميزات**:
- بحث مباشر في العناوين والأوصاف
- فلترة حسب التصنيف
- عداد النتائج

---

### 3. صفحة تفاصيل الدورة (Course Details)
**الوصول**: `#course-1` (أو أي رقم دورة)

**المحتويات**:
- قسم Hero مع معلومات الدورة
- معلومات شاملة (عدد الدروس، المدة، الطلاب، الشهادة)
- أهداف التعلم (Learning Objectives)
- منهج الدورة الكامل (Course Curriculum)
  - أقسام قابلة للتوسيع
  - قائمة الدروس مع المدة
  - دروس معاينة مجانية
- معلومات المدرب
- مميزات الدورة
- أزرار الشراء والتسجيل

**المميزات**:
- منهج قابل للتوسيع والطي
- Sticky sidebar على Desktop
- تصميم متجاوب كامل

---

### 4. مشغل الفيديو (Video Lesson Player)
**الوصول**: `#video-player`

**المحتويات**:
- شريط علوي مع شعار المنصة واسم الدورة
- مشغل فيديو بشاشة كاملة
- أدوات التحكم:
  - تشغيل/إيقاف
  - صوت/كتم
  - شريط التقدم
  - الإعدادات
  - ملء الششاشة
- قائمة جانبية بجميع الدروس
- مؤشر التقدم الكلي
- أزرار التنقل (الدرس السابق/التالي)
- زر "تم إكمال الدرس"

**المميزات**:
- واجهة خالية من التشتيت
- قائمة دروس قابلة للإخفاء/الإظهار
- تتبع الدروس المكتملة
- شريط تقدم ديناميكي
- تصميم responsive كامل

---

### 5. تسجيل الدخول/التسجيل (Login/Register)
**الوصول**: `#login` أو `#register`

**المحتويات**:
- نموذج تبديلي (Login/Register Tabs)
- حقول نموذج التسجيل:
  - الاسم الكامل (للتسجيل فقط)
  - البريد الإلكتروني
  - رقم الجوال (للتسجيل فقط)
  - كلمة المرور (مع إظهار/إخفاء)
- خيار "تذكرني"
- رابط "نسيت كلمة المرور"
- موافقة على الشروط والأحكام
- تسجيل دخول عبر وسائل التواصل:
  - Google
  - Facebook

**المميزات**:
- تصميم نظيف وبسيط
- جاهز لتدفق OTP في المراحل اللاحقة
- تحقق من صحة النموذج
- خلفية gradient جذابة

---

### 6. لوحة الإدارة (Admin Dashboard)
**الوصول**: `#admin`

**المحتويات**:
- شريط جانبي قابل للطي مع التنقل
- لوحة التحكم الرئيسية:
  - إحصائيات (الدورات، الطلاب، الإيرادات، الدروس)
  - جدول الدورات الأخيرة
- إدارة الدورات:
  - جدول بجميع الدورات
  - حالة كل دورة (منشور/مسودة)
  - أزرار التحرير والحذف
- رفع دورة جديدة:
  - معلومات الدورة (العنوان، الوصف، السعر، التصنيف)
  - رفع صورة الدورة
  - رفع دروس الفيديو (ملفات متعددة)
  - حفظ كمسودة أو نشر

**أقسام التنقل**:
1. لوحة التحكم
2. الدورات
3. رفع دورة جديدة
4. الطلاب
5. التقارير
6. الإعدادات

**المميزات**:
- واجهة إدارة احترافية ونظيفة
- شريط جانبي responsive
- جداول بيانات قابلة للفرز
- نماذج رفع ملفات Drag & Drop
- إحصائيات مرئية

## المميزات التقنية

### RTL Support
- جميع الصفحات تدعم اتجاه RTL (من اليمين لليسار)
- تخطيط كامل بالعربية

### Responsive Design
- تصميم mobile-first
- نقاط توقف (Breakpoints):
  - Mobile: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px

### Accessibility
- تباين ألوان واضح
- حجم خط قابل للقراءة
- عناصر تفاعلية واضحة
- Focus states على جميع العناصر

### Performance
- صور محسنة من Unsplash
- تحميل lazy للصور
- انتقالات سلسة ومدروسة
